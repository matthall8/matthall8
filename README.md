## 🔨 What I've built
 
**[street-garms-vlm-classification](https://github.com/matthall8/street-garms-vlm-classification)** — A controlled three-model comparison (zero-shot CLIP vs fine-tuned EfficientNet vs fine-tuned Florence-2; winner at **75.8% macro-F1**, statistically significant), taken from research prototype to a production serverless GPU service with confidence-based abstention derived from beam-search log-probabilities. Projected **£9,500/year saved** for the business.
 
**[aeo-intelligence-agent](https://github.com/NomadicDigital/aeo-intelligence-agent)** — a multi-agent lead-generation system that discovers businesses with weak AI-search visibility, diagnoses their Answer Engine Optimisation gaps and generates actionable recommendations. Live and in use for lead generation at Nomadic Digital.
 
**[ml_from_scratch](https://github.com/matthall8/ml_from_scratch)** — deep learning from first principles in NumPy: attention, tokenisation and Transformer training, plus CNNs and MLPs. Because using LLMs is easier when you understand how they're built underneath.
